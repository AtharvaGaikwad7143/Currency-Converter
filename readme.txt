No server or special setup is required!ğŸ’¡ How It WorksThe magic happens primarily within app.js by orchestrating the following steps:Currency List & Dropdown Population:The codes.js file holds all supported currency codes.app.js loops through this data to dynamically populate both "From" and "To" currency dropdowns on page load. (Default selections are USD $\rightarrow$ INR).Flag Update System:Each currency is mapped to its ISO country code in codes.js.When a currency is selected, JavaScript uses this country code to generate a flag image URL (via FlagsAPI) and instantly update the image next to the dropdown.Fetching Live Exchange Rates:The conversion rate is fetched from the Fawaz Ahmed Currency API.app.js constructs the API URL dynamically based on the user's "From" and "To" currency selections (e.g., .../usd/inr.json).The JSON data returned contains the current conversion rate, which is then used to calculate and display the final converted amount in the message section (.msg).Input Validation:To prevent errors, the input amount is validated.
